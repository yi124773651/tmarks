# 快照功能使用说明

## 快照显示位置

快照功能在两个地方可以访问：

### 1. 浏览器扩展 - 创建快照

**位置**: 浏览器扩展 Popup 界面

**步骤**:
1. 点击浏览器工具栏的扩展图标
2. 在保存书签界面，勾选 **"创建快照"** 复选框
3. 点击 **"保存书签"**
4. 扩展会自动捕获当前页面的 HTML 并上传到服务器

```

│                             │
│ [保存书签]                  │
└─────────────────────────────┘
```

### 2. Web 应用 - 查看和管理快照

**位置**: Web 应用书签卡片

**步骤**:
1. 打开 Web 应用 (https://your-domain.com)
2. 找到有快照的书签（卡片底部会显示"快照"按钮）
3. 点击 **"快照"** 按钮
4. 在弹出的模态框中查看所有快照版本

## 快照访问方式

### 方式 1: 通过 Web 应用界面（推荐）

1. 登录 Web 应用
2. 找到书签卡片
3. 点击"快照"按钮
4. 选择要查看的版本
5. 点击"查看"图标在新窗口打开

**优点**:
- 可以看到所有历史版本
- 可以对比不同版本
- 可以下载或删除快照

### 方式 2: 直接通过 URL 访问

如果你知道快照的 ID，可以直接访问：

```
https://your-domain.com/api/v1/bookmarks/{bookmarkId}/snapshots/{snapshotId}
```

**示例**:
```
https://tmarks.669696.xyz/api/v1/bookmarks/a0eebc99-9c0b-4ef8-bb6d-6bb9bd380a11/snapshots/snap_123456
```

**优点**:
- 直接访问，无需登录
- 可以分享给他人
- 适合嵌入到其他应用

## 常见问题

### Q: 快照会占用多少存储空间？

A: 取决于网页大小：
- 简单网页: 50-200 KB
- 普通网页: 200-500 KB
- 复杂网页: 500 KB - 2 MB
- 使用 SingleFile 后: 可能更大（包含图片等资源）

### Q: 快照会自动删除吗？

A: 是的，系统会自动清理：
- 默认保留最新 5 个版本
- 超过数量时自动删除最旧的
- 可以在设置中调整保留数量

### Q: 快照包含哪些内容？

A: 当前版本：
- ✅ HTML 结构
- ✅ 内联样式
- ❌ 外部 CSS（计划集成 SingleFile）
- ❌ 图片（计划集成 SingleFile）
- ❌ JavaScript

### Q: 如何查看已删除网页的快照？

A: 
1. 在 Web 应用中找到该书签
2. 点击"快照"按钮
3. 选择想要查看的版本
4. 点击"查看"图标

### Q: 可以分享快照给别人吗？

A: 可以，有两种方式：
1. 下载快照为 HTML 文件，发送给对方
2. 分享快照 URL（需要对方有访问权限）

### Q: 快照会包含登录后的内容吗？

A: 不会。快照只捕获当前浏览器看到的 HTML，不包含：
- Cookie
- Session
- 登录状态
- 动态加载的内容（AJAX）
